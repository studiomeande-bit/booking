셀렉 테스트 페이지

실제 세션 ID를 넣고 Netlify 셀렉 페이지를 바로 열어보는 용도입니다. 신규 제출, 수정 제출, 성공 화면, 인보이스 번호 노출까지 빠르게 점검할 수 있습니다.

세션 열기

셀렉 메일에 포함된 `id=` 값만 넣으면 됩니다.

https://select.studio-mean.com/?test=1&id=
테스트 접속 URL
https://select.studio-mean.com/?test=1&id=세션ID

테스트 모드에서는 고객 메일만 발송하지 않습니다. 저장과 관리자 알림은 그대로 유지됩니다.

점검 항목

  • 세션 진입 후 고객명, 상품명, 기본 보정 장수가 정상 표시되는지
  • 기존 제출 데이터가 있으면 복원되는지
  • 추가 인화/추가 보정 비용이 리뷰와 성공 화면에 맞게 보이는지
  • 제출 후 인보이스 번호와 드라이브 링크가 성공 화면에 보이는지
  • 수정 제출 모드에서 다시 제출이 가능한지
