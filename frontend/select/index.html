Photo Selection

이 페이지는 Netlify 정적 프론트의 셀렉 API 연결 버전입니다. 세션 링크 진입, 기존 제출 복원, 제출/수정 제출 경로를 먼저 분리합니다.

세션 정보를 불러오는 중입니다.

세션 정보

URL의 `id` 값을 기준으로 셀렉 세션을 조회합니다.

세션 로딩 전입니다.

마케팅 동의

예약 시 이미 동의한 경우 그대로 표시합니다.

보정 사진 선택

기존 셀렉 폼의 핵심 입력만 우선 분리합니다.

추가 인화

미보정 추가 인화 주문내역 복원과 제출 경로를 함께 검증합니다.

제출

신규 제출 또는 수정 제출이 세션 상태에 따라 자동 전환됩니다.
