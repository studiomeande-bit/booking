예약 시스템 분리 작업 중

이 페이지는 Netlify 정적 프론트의 예약 API 연결 버전입니다. 이번 단계에서는 상품 조회, 날짜/시간 조회, 예약 제출 경로를 먼저 분리합니다.

초기 데이터를 불러오는 중입니다.

1. 상품 선택

Apps Script `getInitDataCustomer()`를 JSON API로 호출합니다.

2. 날짜 선택

상품을 먼저 선택하세요.

-
상품 선택 후 달력을 불러옵니다.

3. 시간 선택

날짜를 선택하면 예약 가능 시간이 표시됩니다.

아직 선택된 날짜가 없습니다.

4. 고객 정보

현재 단계는 기본 예약 제출 경로만 연결합니다.
